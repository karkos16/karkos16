## 🚀 Featured Project

### 🥔 Potato Leaf Disease Detection

A **Jetpack Compose Multiplatform application** that detects potato leaf diseases using a deep learning model.

**Key features:**

* 📷 Detects potato leaf diseases from images
* 🤖 Model based on **EfficientNet Lite** (fine-tuned)
* 📱 Built with **Jetpack Compose Multiplatform**

The goal of this project is to explore how **computer vision models can be integrated directly into mobile applications** to support agriculture and plant disease detection.
